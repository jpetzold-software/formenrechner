Zur Kompilierung des Projektes muss es in einer IDE (Bsp. Eclipse) geöffnet werden. 
Dazu muss das Archiv "KörperFormenRechner" in den workspace von Visual Studio verschoben und entpackt werden.
Dort sollte das C#-Projekt nun angezeigt werden.
 
Um es zu kompilieren und auszuführen gibt es 2 Varianten:
1. die "Veröffentlichung" welche mit einem Rechtsklick auf das Projekt durchgeführt werden kann, dabei wird eine .exe erstellt.
2. die direkte Ausführung in der Entwicklungsumgebung durch Betätigen des "Play"-Buttons.

Um das ausführbare Programm zu starten muss lediglich die bereitgestellte .exe-Datei ausgeführt werden.
Optional kann man dafür auch die selbst erstellte .exe-Datei ausführen.

WICHTIG!
Die im Projekt benutzten Pfade müssen an den Nutzer angepasst werden, ansonsten ist ein Export in XML nicht möglich!
Die Pfade sollten auf den Desktop des Nutzers führen, damit dort Dateien erstellt werden können.
